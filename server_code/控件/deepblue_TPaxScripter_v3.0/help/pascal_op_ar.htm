paxPascal Arithmetic Operators

Language Element Short Description
Multiplication Operator (*) Multiplies two numbers.
Division Operator (/) Divides two numbers and returns a floating-point result.
Integer Division Operator (Div) Divides two numbers and returns an integer result.
Mod Operator (Mod) Divides two numbers and returns only the remainder.
Addition Operator (+) Sums two numbers.
Subtraction Operator (-) Finds the difference between two numbers or indicates the negative value of a numeric expression.

Multiplication Operator (*)
Multiplies two numbers. 
result = number1*number2

Arguments

result
Any numeric variable
number1, number2
Any numeric expression
If one or both expressions are Null expressions, result is Null.

Division Operator (/)
Divides two numbers and returns a floating-point result. 
result = number1/number2

Arguments

result
Any numeric variable
number1, number2
Any numeric expressions
If one or both expressions are Null expressions, result is Null.
