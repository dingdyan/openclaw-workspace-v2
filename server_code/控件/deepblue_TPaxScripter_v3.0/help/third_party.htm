Third-Party Tools

  • PaxImporterD7 component (freeware). Author: Jacques Garcia.
    The component allows you to customize your imp files and to produce the batch conversion of files directly in the list view. The distribution package includes an advanced demo application. Click here to download PaxImporterD7 sources (60Kb).
  • TPaxModuleScripter (freeware). Author: Maxim Kuznetsov.
    The component gives you an easy way of importing Delphi forms including event handlers. Click here to download TPaxModuleScripter sources (6Kb).
  • IWPaxScript (freeware). Author: Alain Falanga.
    IWPaxScript is a component for IntraWeb (a web application development framework for Visual Studio.NET, Delphi, C++ Builder, Kylix, and Java) that allows to use paxScript engine inside an IntraWeb application. Click here to download IWPaxScript sources (16Kb).
  • paxDebuggershareware) (Era Soluzioni S.r.l.).
    paxDebugger is a VBA like environment, where script extends application, and make it customizable by the end-user, without deploy the whole developer environment (ex: Delphi). This impressive component is developed and supported by Era Soluzioni S.r.l . The paxDebugger is an open source project with a GNU Lesser General Public License.
  • TPaxWeScripter and TPaxPageProducer components (freeware). Author: Don Wibier, Any Key Software Solutions.
    • TPaxWebScripter works in the same way as a Delphi TPageProducer with the major difference that it will enable scripting in your web templates just like the Active Server Pages.
    • TPaxPageproducer is a bridge between the PageProducer system of Borland and the TpaxWebScripter. It can be used in the same way as the Borland PageProducers with the difference that scripts support languages supported by the TPaxScripter.
    Click here to download the PaxWebScripter package.
  • TPaxScripter help, chm and hlp formats. Author: Chris Campbell.
  • paxScript Importer (freeware). Author: Matt Harward.
    paxScript Importer allows you to import Delphi units automatically. The Importer parses a Delphi unit and generates the imp unit, so you can register all constants, routines and classes of the Delphi unit for the paxScript engine in a few seconds!
