paxC Logical Operators

Language Element Short Description
And AND Operator (&&) Performs a logical conjunction on two expressions.
Logical NOT Operator (!) Performs logical negation on an expression.
Logical OR Operator (!!) Performs a logical disjunction on two expressions.

And AND Operator (&&)
Performs a logical conjunction on two expressions. 
result = expression1 && expression2

Arguments

result
Any numeric expression
expression1, expression2
Any expressions
If one or both expressions are Null expressions, result is Null.

Logical NOT Operator (!)
Performs logical negation on an expression. 
result = !expression

Arguments

result
Any numeric expression.
Expression
Any expression
If Expression is Null, result is Null.

Logical OR Operator (!!)
Performs a logical disjunction on two expressions. 
result = expression1 !! expression2

Arguments

result
Any numeric expression
expression1, expression2
Any expressions
If one or both expressions are Null expressions, result is Null.
