paxC Comparison Operators

Language Element Short Description
Operator (<) Returns a Boolean value indicating the result of the comparison "less than".
Operator (<=) Returns a Boolean value indicating the result of the comparison "less than or equal to".
Operator (>) Returns a Boolean value indicating the result of the comparison "greater than".
Operator (>=) Returns a Boolean value indicating the result of the comparison "greater than or equal to".
Operator (=) Returns a Boolean value indicating the result of the comparison "equal to".
Operator (!=) Returns a Boolean value indicating the result of the comparison "not equal to".

Operator (<)
Returns a Boolean value indicating the result of the comparison "less than". 
result = expression1 < expression2

Arguments

result, expression1, expression2
Any expression
If one or both expressions are Null expressions, result is Null.
