Variant

You can omit declaration of Variant types in your paxBasic scripts. For example the following script 

Function F(X)
  Dim Y = rnd(0, 1000)
  return X + Y + Y
End Function
is equivalent to 
Function F(X As Variant) As Variant
  Dim Y As Variant = rnd(0, 1000)
  return X + Y + Y
End Function
