You can omit declaration of Variant types in your paxPascal scripts. For example the following script 

function MyFunc(X);
var Y;
begin
  Y := Random(100);
  result := X + Y + Y;
end;
print MyFunc(3);
is equivalent to 
function MyFunc(X: Variant): Variant;
var Y: Variant;
begin
  Y := Random(100);
  result := X + Y + Y;
end;
print MyFunc(3);
