paxPascal Miscellaneous Operators

Language Element Short Description
@ Operator Returns delegate of function or alias of variable.
^ Operator Returns terminal of variable.
+ Operator (unary) Returns copy of array.

@ Operator
Returns delegate of function or alias of variable (see LISPPA ). 
result = @ Expr

Arguments

result
Any variable.
Expr
Any expression.

^ Operator
Returns terminal of variable (see LISPPA ). 
result = Expr ^

Arguments

result
Any variable.
Expr
Any expression.

+ Operator (unary)
Returns copy of array. 
result = + expr

Arguments

expr
Any expression
